# AI 人工智慧与机器学习专有名词

## 语言模型相关 (Language Models)

- **LLM**: Large Language Model (大型语言模型)
- **LMM**: Large Multimodal Model (大型多模态模型)
- **MLM**: Medium Language Model (中型语言模型)
- **SLM**: Small Language Model (小型语言模型)
- **LRM**: Large Reasoning Model (大型推理模型)
- **VLM**: Vision-Language Model (视覺语言模型)
- **TTS**: Text-to-Speech (语音合成)
- **STT/ASR**: Speech-to-Text / Automatic Speech Recognition (语音辨识)

## 检索增强与推理 (Retrieval & Reasoning)

- **RAG**: Retrieval-Augmented Generation (检索增强生成)
- **RAT**: Retrieval-Augmented Thinking (检索增强思維)
- **CoT**: Chain of Thought (思維鏈)
- **ToT**: Tree of Thoughts (思維樹)
- **Prompt Engineering**: 提示词工程
- **System Prompt**: 系统提示词

## 模型架构 (Model Architectures)

- **MoE**: Mixture of Experts (专家混合模型)
- **Transformer**: 變形金剛架构 (注意力机制核心)
- **Attention Mechanism**: 注意力机制 (Self-Attention)
- **GNN**: Graph Neural Network (图神经网路)
- **CNN**: Convolutional Neural Network (卷積神经网路)
- **RNN**: Recurrent Neural Network (循环神经网路)
- **LSTM**: Long Short-Term Memory (长短期记忆网路)
- **GRU**: Gated Recurrent Unit (門控循环单元)

## 訓练与学习技术 (Training & Learning)

- **RLHF**: Reinforcement Learning with Human Feedback (帶人类回馈的强化学习)
- **RLAIF**: Reinforcement Learning from AI Feedback (帶 AI 回馈的强化学习)
- **SFT**: Supervised Fine-Tuning (监督式微调)
- **Pre-training**: 预訓练
- **Fine-Tuning (FT)**: 微调
- **LoRA**: Low-Rank Adaptation (低秩适应微调)
- **Supervised Learning (SL)**: 监督学习
- **Unsupervised Learning (UL)**: 非监督学习
- **Self-Supervised Learning (SSL)**: 自监督学习
- **Overfitting**: 过擬合 / 过度擬合
- **Underfitting**: 欠擬合 / 擬合不足

## 数据与NLP处理 (Data & NLP)

- **NLP**: Natural Language Processing (自然语言处理)
- **CV**: Computer Vision (电腦视覺)
- **OCR**: Optical Character Recognition (光学字元辨识)
- **NER**: Named Entity Recognition (命名实体辨识)
- **Tokenization**: 分词 / Token 化
- **Vector Embedding**: 向量嵌入
- **Vector Database**: 向量资料库 (Chroma, Pinecone, Milvus)
- **ETL**: Extract, Transform, Load (提取、转换、载入)

## 机器学习基础 (General ML)

- **AI**: Artificial Intelligence (人工智慧)
- **ML**: Machine Learning (机器学习)
- **DL**: Deep Learning (深度学习)
- **AGI**: Artificial General Intelligence (通用人工智慧)
- **ASI**: Artificial Superintelligence (超人工智慧)
- **Neural Network**: 类神经网路
- **Weights & Biases**: 权重与偏差

## 储存、框架与计算 (Frameworks & Hardware)

- **PyTorch**: (Meta 开源的深度学习框架)
- **TensorFlow**: (Google 的深度学习框架)
- **Hugging Face**: (开源 AI 社群与模型库平台)
- **Ollama**: (本地执行大模型的轻量框架)
- **vLLM**: (高吞吐量的大模型推理与服务引擎)
- **GPU**: Graphics Processing Unit (图形处理单元)
- **TPU**: Tensor Processing Unit (張量处理单元)
- **NPU**: Neural Processing Unit (网絡处理单元)
- **CUDA**: NVIDIA 显示卡并行计算架构
- **ONNX**: Open Neural Network Exchange (开放神经网路交换格式)
