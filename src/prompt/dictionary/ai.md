# AI 人工智慧與機器學習專有名詞

## 語言模型相關 (Language Models)

- **LLM**: Large Language Model (大型語言模型)
- **LMM**: Large Multimodal Model (大型多模態模型)
- **MLM**: Medium Language Model (中型語言模型)
- **SLM**: Small Language Model (小型語言模型)
- **LRM**: Large Reasoning Model (大型推理模型)
- **VLM**: Vision-Language Model (視覺語言模型)
- **TTS**: Text-to-Speech (語音合成)
- **STT/ASR**: Speech-to-Text / Automatic Speech Recognition (語音辨識)

## 檢索增強與推理 (Retrieval & Reasoning)

- **RAG**: Retrieval-Augmented Generation (檢索增強生成)
- **RAT**: Retrieval-Augmented Thinking (檢索增強思維)
- **CoT**: Chain of Thought (思維鏈)
- **ToT**: Tree of Thoughts (思維樹)
- **Prompt Engineering**: 提示詞工程
- **System Prompt**: 系統提示詞

## 模型架構 (Model Architectures)

- **MoE**: Mixture of Experts (專家混合模型)
- **Transformer**: 變形金剛架構 (注意力機制核心)
- **Attention Mechanism**: 注意力機制 (Self-Attention)
- **GNN**: Graph Neural Network (圖神經網路)
- **CNN**: Convolutional Neural Network (卷積神經網路)
- **RNN**: Recurrent Neural Network (循環神經網路)
- **LSTM**: Long Short-Term Memory (長短期記憶網路)
- **GRU**: Gated Recurrent Unit (門控循環單元)

## 訓練與學習技術 (Training & Learning)

- **RLHF**: Reinforcement Learning with Human Feedback (帶人類回饋的強化學習)
- **RLAIF**: Reinforcement Learning from AI Feedback (帶 AI 回饋的強化學習)
- **SFT**: Supervised Fine-Tuning (監督式微調)
- **Pre-training**: 預訓練
- **Fine-Tuning (FT)**: 微調
- **LoRA**: Low-Rank Adaptation (低秩適應微調)
- **Supervised Learning (SL)**: 監督學習
- **Unsupervised Learning (UL)**: 非監督學習
- **Self-Supervised Learning (SSL)**: 自監督學習
- **Overfitting**: 過擬合 / 過度擬合
- **Underfitting**: 欠擬合 / 擬合不足

## 數據與NLP處理 (Data & NLP)

- **NLP**: Natural Language Processing (自然語言處理)
- **CV**: Computer Vision (電腦視覺)
- **OCR**: Optical Character Recognition (光學字元辨識)
- **NER**: Named Entity Recognition (命名實體辨識)
- **Tokenization**: 分詞 / Token 化
- **Vector Embedding**: 向量嵌入
- **Vector Database**: 向量資料庫 (Chroma, Pinecone, Milvus)
- **ETL**: Extract, Transform, Load (提取、轉換、載入)

## 機器學習基礎 (General ML)

- **AI**: Artificial Intelligence (人工智慧)
- **ML**: Machine Learning (機器學習)
- **DL**: Deep Learning (深度學習)
- **AGI**: Artificial General Intelligence (通用人工智慧)
- **ASI**: Artificial Superintelligence (超人工智慧)
- **Neural Network**: 类神經網路
- **Weights & Biases**: 權重與偏差

## 儲存、框架與計算 (Frameworks & Hardware)

- **PyTorch**: (Meta 開源的深度學習框架)
- **TensorFlow**: (Google 的深度學習框架)
- **Hugging Face**: (開源 AI 社群與模型庫平台)
- **Ollama**: (本地執行大模型的輕量框架)
- **vLLM**: (高吞吐量的大模型推理與服務引擎)
- **GPU**: Graphics Processing Unit (圖形處理單元)
- **TPU**: Tensor Processing Unit (張量處理單元)
- **NPU**: Neural Processing Unit (網絡處理單元)
- **CUDA**: NVIDIA 顯示卡並行計算架構
- **ONNX**: Open Neural Network Exchange (開放神經網路交換格式)
