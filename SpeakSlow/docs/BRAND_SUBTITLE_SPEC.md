# 品牌副標題規格文檔

## 概述

本文檔定義「聲聲慢」品牌副標題的使用規範，以及在 Electron 應用中的實作方式。

## 品牌核心概念

- **品牌名稱**：聲聲慢
- **核心對比**：「聲聲慢，轉錄快」
- **品牌精神**：以詩意的名稱對比極速的效能

## 副標題選項

### 選項一（推薦）
**聲聲慢 - 極速語音轉錄**
- 優點：簡潔有力，直接傳達核心價值
- 適用：所有窗口標題

### 選項二
**聲聲慢 - 最快的語音轉錄軟體**
- 優點：強調市場定位
- 缺點：標題較長

### 選項三
**聲聲慢 - 聲聲慢，轉錄快**
- 優點：完整呈現品牌對比概念
- 缺點：重複品牌名稱

## Electron Window Title 格式

### 主窗口（Main Window）
- 無標題列 (frame: false)
- 不需設定 title，因為視窗沒有標題列

### 控制面板窗口（Control Panel Window）
```
聲聲慢 - 極速語音轉錄
```

### 歷史記錄窗口（History Window）
```
轉錄歷史 - 聲聲慢
```
- 保持現有格式，功能名稱在前

### 設定窗口（Settings Window）
```
設定 - 聲聲慢
```
- 保持現有格式，功能名稱在前

## 需要修改的檔案清單

### 1. `src/helpers/windowManager.js`
- 新增控制面板窗口的 title 屬性
- 確認歷史記錄和設定窗口的標題格式

### 2. `package.json`
- 確認 `build.productName` 為「聲聲慢」（已設定）

```

```
